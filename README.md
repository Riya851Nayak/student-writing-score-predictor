# Student Writing Score Predictor

This is a Machine Learning web application built using **Streamlit**.

## Project Overview
The app predicts a student's **writing score** based on:
- Math score
- Reading score

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
